# Halden Home Climate

A fictional, one-page portfolio concept for a premium residential heating, cooling, and indoor-air-quality company serving Phoenix, Scottsdale, and the East Valley.

## Local development

```bash
npm install
npm run dev
```

## Verification

```bash
npm run lint
npx tsc --noEmit
npm test
```

The project is a React and TypeScript site built with Vite through vinext. It has no backend, database, authentication, or form transmission. All photography and fonts are bundled locally. The site includes `noindex, nofollow` metadata because the company is fictional.
